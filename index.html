Comment fonctionne la sélection

01 · Univers

187 actions issues du CAC40, DAX40, AEX25 et S&P100. Filtrées chaque lundi par un script Python tournant sur GitHub Actions.

02 · Scoring

Chaque action reçoit un score /100 : momentum technique (MM50, MM200, RSI, volume) + fondamentaux (croissance CA, marges, PEG) + consensus analystes.

03 · Transparence

La méthodologie, le code source et l'historique des changements sont publics sur GitHub. Les 25 meilleures scores composent la watchlist.
